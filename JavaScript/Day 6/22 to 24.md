# 🌐 JavaScript DOM Tutorial

## 📘 1. What is the DOM?

- **DOM** stands for **Document Object Model**.
- It represents the **HTML document as a tree structure** where each node is an object representing a part of the document.

### Key Concepts:
- DOM is a **programming interface** for HTML and XML documents.
- Allows **JavaScript** to manipulate elements, attributes, and content.

```html
<!-- Sample HTML -->
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a paragraph.</p>
  </body>
</html>


🌳 2. DOM Tree Structure
The above HTML is represented in the DOM as a tree of nodes:

css
Copy
Edit
Document
 └── html
     ├── head
     │   └── title
     │       └── "My Page"
     └── body
         ├── h1
         │   └── "Hello World"
         └── p
             └── "This is a paragraph."


Types of DOM Nodes:
Element Nodes – HTML elements (<h1>, <p>, etc.)

Text Nodes – Text inside elements

Attribute Nodes – Attributes of elements (e.g., id, class)

🛠️ 3. Accessing the DOM in JavaScript
JavaScript provides several methods to interact with the DOM:


// Get element by ID
const heading = document.getElementById("myHeading");

// Get elements by tag name
const paragraphs = document.getElementsByTagName("p");

// Get elements by class name
const boxes = document.getElementsByClassName("box");

// Modern query selectors
const element = document.querySelector(".container");
const allElements = document.querySelectorAll("div.box");
⚡ 4. Introduction to Events in JavaScript
Events are actions or occurrences that happen in the browser (e.g., click, hover, keypress).

JavaScript can respond to these events using event handlers.

Common Events:
Event	Description
click	Mouse click on an element
mouseover	Mouse hovers over an element
keydown	Key is pressed on keyboard
submit	Form is submitted

Adding Event Listeners:
js
Copy
Edit
const btn = document.getElementById("myBtn");

btn.addEventListener("click", function () {
  alert("Button clicked!");
});
The addEventListener() method allows multiple event handlers on the same element.

📚 Summary
The DOM allows dynamic access to HTML content using JavaScript.

The DOM Tree represents the hierarchical structure of the page.

Events enable interaction between the user and the webpage.
